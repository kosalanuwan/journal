Writing vanilla JS snippets and examples

10th February 2012
2 minute read
Tweet

JavaScript is obnoxious! Do you think that’s because it is based on miserably failed languages - Java, Scheme, and Self - at that time, or do you think it’s because Netscape did not give enuf time for Eich to design, implement, and ship JavaScript? For the record, you can make things work with absolutely no idea of what you are doing in JavaScript.

The truth is, I’m getting all kinds of help from our friends at JS and VS Code community and I need a library of helper functions and boilerplate code written with vanilla JS. Easy enuf, except I need a playground without having to install NVM and Node.js like they do on Codepen. It gives the whole thing a very Control Freakery feel since the development environment will be more implicit than, say, adhering to frameworks and tools to build and run from source.

The first thing we’re going to need is a modern web browser that ship with Developer Tools. I can use the Snippets and the Console for coding and debugging so a little copy paste magic is all that is necessary to get started with vanilla JS. Kids’ stuff.

Next is a free web host running locally. I’m not gonna lie, you’d be amazed how tricky it is to pick a node-based web server. Not only are there many frameworks like Fastify, but there are node modules with VS Code extensions like Live Server. There’s no way I’m going to do npm init fastify boilerplate code one at a time. So, it’s definitely necessary to break out .devcontainer.json and add the live server extension. Then I can create JS snippets with example static sites and the live server will preview on localhost for me. Excellent.

Here’s the library of vanilla JS snippets and examples I pulled together so far.

  1. Huges, D. (2014). Douglas Crockford Lectures on JavaScript [Video playlist].
  2. Fog City Learning. (2017). DevTools Basics: Chrome Developer Tools Tutorial [Video playlist].
  3. Codes, J. (2020). How to set up Live Server and browser auto refresh in VS Code [Video].
  4. Mutunhire, T. (2018). How to start a node server: examples with the most popular frameworks [Blog] Stack abuse.
