Personal productivity

5th November 2015
1 minute read
Tweet

Here’s my deliberate efforts to keep dancing. These tools and apps are my personal preference and probably might work for you as well. Not going to lie, I may have literally let go of some of these for the preference of my employers and clientele.

Contents

  • Notes written in markdown using Paper app
  • Task tracking using GitHub
  • Todo list of ahaa moments using GitHub app
  • Reminders are always on a calendar
  • Emails are filtered using Outlook rules
  • Reading as in books are only on Audible app
  • Budgets are planned monthly using Google Keep

Notes

  • Written in markdown using Paper app

Previously used OneNote, Evernote, Workflowy, Dropbox Paper, and even Notion.

Task tracking

  • GitHub Issues for task and bug tracking
  • Milestones for road maps and sprint planning, and
  • Labels to categorize based on features, bugs, and whatnot

This is to keep a track of todos on various projects. Not meant to plan my daily routines. Previously used Trello, Todoist, and even Asana.

Todo list

  • Todos repository on GitHub
  • GitHub Issues for each todo item

It seems like a pretty fantastic idea in lieu of traditional Trello, Todoist, Microsoft Todo, Google Keep, and the like. You want to try out GitHub app on the mobile.

Reminders

  • Google Calendar

Previously tried Workflowy, Microsoft Todo, Trello, but it has always been a calendar when scheduling meetings, appointments, reminders et al. whether it’s personal or work.

Emails

  • Microsoft Outlook Rules to filter and process emails and meeting invites

You want to have a look at Scott Hanselman’s article on Outlook rules for processing emails.

Reading

  • Audio books purchased from Audible
  • Listens using Audible app while on the go

Credit goes to former colleague and a dear friend, @hasith, for showing how to keep up using Audible. I’m a snail-paced reader, still reads a chapter or two occasionally to get inspiration for fiction writing.

Budgets

  • Google Keep as a simple list of expenses with the budgeted amount

Previously tried many apps and even spreadsheets but that doesn’t seemed to work consistently for not more than a few months.
